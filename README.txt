Student Mental Health System
Instructions:
1. Move the extracted folder to your XAMPP htdocs directory (e.g., C:\xampp\htdocs\student_mental_health_system)
2. Create the database by importing the file 'sql/create_db.sql' using phpMyAdmin or MySQL CLI.
3. Update includes/config.php if your DB credentials are different.
4. Start Apache and MySQL in XAMPP, then visit:
   - http://localhost/student_mental_health_system/register.php  (create student)
   - http://localhost/student_mental_health_system/login.php     (student login)
   - http://localhost/student_mental_health_system/admin/login.php (admin login)
Admin default credentials (after running SQL import):
   username: admin
   password: adminpass
